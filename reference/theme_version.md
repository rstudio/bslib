# Obtain a theme's Bootstrap version

Obtain a theme's Bootstrap version

## Usage

``` r
theme_version(theme)
```

## Arguments

- theme:

  A
  [`bs_theme()`](https://rstudio.github.io/bslib/reference/bs_theme.md)
  object.

## Value

Returns the major version of Bootstrap used in the `theme`.
