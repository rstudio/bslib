# Obtain a theme's Bootswatch theme name

Obtain a theme's Bootswatch theme name

## Usage

``` r
theme_bootswatch(theme)
```

## Arguments

- theme:

  A
  [`bs_theme()`](https://rstudio.github.io/bslib/dev/reference/bs_theme.md)
  object.

## Value

Returns the Bootswatch theme named used (if any) in the `theme`.
